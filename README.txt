・Statistic_resultのデータについて
    今回はOLS（最小二乗法）を用いて線形重回帰をした。OLSの詳しい式については割愛。

・データの見方
    一番左上の'Dep. Variable'が分析対象の名前を表す。

    その少し下から始まるテーブルの列名についてそれぞれ解説すると、
    ・coef -> 係数。これが大きいほど、その指標の変化が分析対象にもたらす変化の大きさが大きい。
    ・std err -> 標準誤差。今回はあまり気にしなくていい。
    ・t -> t値。これも気にしなくていい。
    ・P>|t| -> P値。これが0.05以下ならその指標は分析対象と関係してるとみてよい。（今回は緩めに0.1ぐらいより小さいならアリとしてる）
    ・[0.025, 0.075] -> 第一四分位数と第三四分位数。これも気にしなくていい。
